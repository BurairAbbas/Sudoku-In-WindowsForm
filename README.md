# Sudoku-In-WindowsForm
4x4 sudoku created for OOP project.

### Features:
* 3 levels in the game.
* Difficulty level increase by levels.
* Score board.
* Verify the sudoku Logic at every input.
* Can input value from both button and keyboard.
